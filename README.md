# Bank Management System

A command-line banking application developed in Java that simulates basic banking operations such as account creation, deposits, withdrawals, and balance inquiries.

## Features

- Account Creation
- Deposit Management
- Withdrawal Processing
- Balance Inquiry
- Customer Data Handling
- Input Validation

## Technologies Used

- Java
- File Handling
- Functions and Methods
- Conditional Statements
- Loops

## Learning Outcomes

This project was developed to strengthen understanding of:

- Core Java Programming
- Data Validation
- File Handling
- Modular Programming
- Banking System Logic

## Future Improvements

- Graphical User Interface
- Database Integration
- Transaction History
- Account Security Features
- Multi-User Support
